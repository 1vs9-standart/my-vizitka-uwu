VRC Media Server

web app

Скачивает видео (YouTube, торренты, прямые ссылки), перекодирует в H.264/AAC и отдаёт готовый файл для видеоплеера в VRChat. Решает проблемы с 18+, форматами, геоблоком.

Python FFmpeg yt-dlp Docker
